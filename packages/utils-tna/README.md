## Installation

```sh
npm install utils-tna
```

or using Yarn:

```sh
yarn add utils-tna
```

or using pnpm:

```sh
pnpm add utils-tna
```

## Usage

```ts
import { formatQueryString, convertObjectKeysToSnakeCase } from 'utils-tna'
```
